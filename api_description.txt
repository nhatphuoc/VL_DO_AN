Endpoint: /homeData
Description: Lấy thông tin chung
- Response:
  {
    "foodRemaining": number,        // Lượng đồ ăn còn lại
    "waterRemaining": number,       // Lượng nước còn lại
    "roomTemperature": number,      // Nhiệt độ phòng (Celsius)
    "roomHumidity": number,         // Độ ẩm phòng (%)
    "battery": number,              // Pin (%)     Chưa làm
    "latestImage": {
      "url": string,               // URL của hình ảnh chụp gần nhất
      "time": number          // Thời gian chụp (Unix timestamp)
    }
  }

### Endpoint: gallery
- List environment(POST /gallery/list)
  Request:
  {
    "startDate": number,             // Ngày bắt đầu (Unix timestamp)
    "endDate": number                // Ngày kết thúc (Unix timestamp)
  }
  Response:
  {
    "galleryHistory": [
      {
        "url": string,               // URL của hình ảnh hoặc encode byte64
        "time": number          // Thời gian chụp (Unix timestamp)
      },
      ...
    ]
  }
- Create environment(POST /gallery/create)
  Request:
    {
      "url": string,               // URL của hình ảnh hoặc encode byte64
      "time": number          // Thời gian chụp (Unix timestamp)
    }
  Response:
  true

### Description: Quản lý lịch ăn
- Create schedule(POST /schedule/):
  Request:
    {
        "value": number,               // Lượng thức ăn sẽ cho
        "time": number,                // Thời gian cho ăn (Unix timestamp)
        "isOn": boolean                // Bật hay tắt lịch này
    }

  Response:
    true


- Update schedule(PATCH /schedule/id):
  Request:
    {
        "value": number,               // Lượng thức ăn sẽ cho
        "time": number,                // Thời gian cho ăn (Unix timestamp)
        "isOn": boolean                // Bật hay tắt lịch này
    }

  Response:
    true


- Delete schedule(DELETE /schedule/id):
  Response:
    true

- List schedule(GET /schedule/):

    "feedingSchedule": [
